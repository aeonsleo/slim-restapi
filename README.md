# MongoDB-PHP7-RestAPI Tutorial
A small example of a blog using MongoDB with PHP7.

## It was tested with
1. PHP 7.0.15
2. MongoDB 3.2.13
3. mongodb-driver 1.1.9
4. Apache 2.4.18
